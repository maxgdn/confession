# Confession

Confess to how you feel and what must get done.

## Getting Started


### Prerequisites

- Kotlin
- JDK 11+
- ESC-POS Printer (optional)
- Windows, Linux, MacOS

### Installing

```
git clone https://github.com/maxgdn/confession.git
```

Sync gradle and run the MainKt file.
